# Autonomous Nexus: Comprehensive User Guide v2.0

Welcome to the **Autonomous Nexus Spatial Operating System**. This platform replaces traditional dashboards with an Infinite Canvas, powered by autonomous AI agents (Google Gemini 2.5 Pro) and WebGL visualizations.

## 1. Authentication & Security Clearance

### Account Creation & 2-Step Verification
1.  **Registration:** On the main login screen, select "Register" to create a new Operator account.
2.  **Two-Factor Authentication (TOTP):** Once logged in, immediately navigate to **Security & Profile** in the sidebar. 
    *   Click "Enable 2-Step Verification".
    *   Scan the generated cryptographic QR code using Google Authenticator, Authy, or Microsoft Authenticator.
    *   Enter the 6-digit pin to lock your security settings. Future logins will require this rotating code.

### Subscription Tiers (Security Clearances)
Advanced modules are gated by your security clearance. Navigate to **Billing & Upgrade** to manage your subscription via Stripe.
*   **Free Trial (14 Days):** Basic Recon tools, 1 active AI Agent.
*   **Operative:** Standard Arsenal, Node-Based Playbooks.
*   **Commander:** Unlocks Adaptive Ops (Generative UI) and Non-Euclidean Target Portals.
*   **Nexus Prime:** Unlocks Tier 5 Quantum Cyber Warfare, Headless API Access, and the Global Kill Switch.

## 2. Platform Navigation (The Spatial OS)

*   **The Omnibar:** Press `Cmd + K` (Mac) or `Ctrl + K` (Windows/Linux) to open the Global Command Palette. Type commands like "Recon", "Quantum", or "Billing" to instantly navigate.
*   **Zenith HUD Mode:** Type "Zenith" into the Omnibar to strip away all borders, menus, and sidebars. This enters a distraction-free, edge-to-edge mode. Press `ESC` to exit.
*   **Mobile Gestures:** On iOS/Android, use pinch-to-zoom to navigate 3D models and the persistent bottom navigation bar to switch modules.

*Requires Commander Clearance.*

*Requires Commander Clearance.*

*   **Environments:** You can physically visualize a compromised Database (red data blocks), a vulnerable Web Server (cyan directory wireframes), or an IoT Botnet (amber mesh nodes). Click "Close Portal & Extract" to return to the global view.

### D. Quantum Cyber Warfare

## 4. Enterprise Integrations & Emergency Protocols

### Headless API Access (CI/CD)
*Requires Nexus Prime Clearance.*
Navigate to **API Integrations** to generate cryptographic keys. These keys can be inserted into GitHub Actions, Jenkins, or external scripts. Passing the `X-API-Key` header allows your CI/CD pipelines to trigger autonomous vulnerability scans automatically upon code deployment.

### The Global Kill Switch
*Requires Nexus Prime or Admin Clearance.*
Located in the top right of the application interface is the **Kill Switch**. 
If an autonomous agent operates outside of its designated parameters, click this button to arm it, then click "CONFIRM OVERRIDE".
1.  It instantly halts the global autonomous AI loop.
2.  It force-fails all active and pending missions.
3.  It writes an immutable, tamper-proof record to the SQL Audit Log proving human intervention, securing your legal liability.
